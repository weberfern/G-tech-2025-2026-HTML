# Geração Tech 3.0 - HTML

## Descrição

Este repositório contém as atividades de aprendizado em **HTML** (HyperText Markup Language) do bootcamp **Geração Tech 3.0**, um programa focado no desenvolvimento web com tecnologias front-end e back-end.


## Atividades

### Atividade 01 - Introdução ao HTML

A primeira atividade foca nos fundamentos do HTML, demonstrando a estruturação semântica de uma página web.

**Características principais:**
- **Estrutura semântica**: Utiliza tags HTML semânticas como `<header>`, `<nav>`, `<main>`, `<article>`, e `<section>`
- **Navegação**: Menu de navegação com links internos
- **Conteúdo**: Apresenta informações sobre as principais tags HTML e suas funcionalidades
- **Responsividade**: Meta tag de viewport para adaptação em diferentes dispositivos

**Tecnologias**: HTML5

[Ver detalhes da Atividade 01](Atividade%2001/README.md)

### Atividade 02 - HTML com CSS e Elementos Multimídia

A segunda atividade expande o aprendizado incluindo **CSS inline** para estilização e elementos multimídia (áudio e vídeo).

**Características principais:**
- **Estrutura semântica**: Organização clara do conteúdo utilizando tags HTML semânticas
- **Estilos CSS inline**: Propriedades CSS como `margin`, `padding`, `display: flex`, `gap`, e `list-style-type`
- **Elementos multimídia**: Player de áudio MP3 e vídeo do YouTube incorporado
- **Flexbox**: Uso de `display: flex` para layout responsivo
- **Responsividade**: Adaptação em diferentes dispositivos

**Tecnologias**: HTML5, CSS3

[Ver detalhes da Atividade 02](Atividade%2002/README.md)

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/weberfern/G-tech-2025-2026-HTML.git
```

2. Navegue até a pasta HTML:
```bash
cd "HTML"
```

3. Acesse a atividade desejada:
```bash
cd "Atividade 01"  # ou "Atividade 02"
```

4. Abra o arquivo `index.html` em seu navegador de preferência.
